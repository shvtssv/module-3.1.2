# Калькулятор

Этот проект — очень простой терминальный калькулятор для быстрых арифметических вычислений.
Запускается локально на Windows и macOS, работает без внешних зависимостей.
Имеет функционал истории для отслежки прошлых расчётов.

## Возможности

- Базовые операции: сложение (+), вычитание (-), умножение (\*), деление (/), остаток (%)
- Возведение в степень с помощью оператора ^ (например, 2 ^ 3 = 8)
- История последних вычислений: нажмите `h` в любой момент, чтобы увидеть до 10 последних записей
- Продолжение цепочки вычислений: после результата можно продолжить расчёт, используя результат как левый операнд
- Выход из программы: `q` на любом шаге

## Структура

- `calculator.py` — основной скрипт калькулятора
- `README.md` — описание проекта и инструкции

## Требования

- Python 3.8+ (дополнительных библиотек нет).

## Установка

1. Клонирование репозитория:
   - `git clone <URL_РЕПОЗИТОРИЯ> && cd <название папки с репо>`
2. (Опционально) Создание и активация виртуального окружения:
   - macOS/Linux: `python3 -m venv .venv && source .venv/bin/activate`
   - Windows (PowerShell): `py -m venv .venv; .\.venv\Scripts\Activate.ps1`

## Запуск

cd calculator-project
python3 calculator.py

Пример CLI-сессии:

```
Введите первое число: 2
Введите оператор (+ - * / % ^): *
Введите второе число: 3
Результат: 6.0
Продолжить (Enter/c) | новый расчёт (n) | история (h) | выход (q): q
```

## Как пользоваться

Программа задаёт шаги последовательно:

1. Введите первое число (или нажмите `h` для истории, `q` для выхода)
2. Введите оператор (+ - \* / % ^) — `h` и `q` тоже работают
3. Введите второе число — `h` и `q` тоже работают

После этого отображается результат и появляется вопрос, что делать дальше:

- `c` (или просто Enter): продолжить с полученным результатом
- `n`: начать новый расчёт
- `h`: показать историю расчётов
- `q`: выйти
